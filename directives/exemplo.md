# Diretiva de Exemplo: Executar Soma

## Objetivo
Demonstrar a execução de um script Python simples através da arquitetura do Agente.

## Entradas
- Dois números.

## Ferramentas
- `execution/exemplo.py`

## Procedimento
1.  Verifique se o script `execution/exemplo.py` existe.
2.  Execute o script passando dois números como argumentos via linha de comando.
3.  Capture a saída do script.

## Saída Esperada
- A soma dos dois números impressa no console.
